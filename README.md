# 📚 MERN Bookstore Application

## 🚀 Project Overview
This is a full-stack Bookstore web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). It allows users to browse, add, update, and manage books.

## ✨ Features
- User Authentication (Login/Register)
- Add, Update, Delete Books
- Search and Filter Books
- Responsive UI
- REST API integration

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Others: Axios, JWT, Bootstrap / Tailwind (if used)
